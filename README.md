# 可拖动刷新按钮 Chrome 扩展

一个简单的 Chrome 浏览器扩展，在页面上添加一个可拖动的按钮，点击后执行硬刷新操作。

## 功能特性

- 🎯 可拖动的圆形按钮
- 🔄 点击按钮执行硬刷新（忽略缓存）
- 📱 支持桌面和移动设备
- 🎨 美观的悬停和点击效果
- 🔍 自动适应单页应用
- 💾 记忆按钮位置（使用localStorage）

## 安装方法

1. 打开 Chrome → 进入 `chrome://extensions/`
2. 开启"开发者模式"
3. 选择一种方式：
   - 点击"加载已解压的扩展程序"，选择扩展文件夹
   - 或将 `refresh-button.zip` 拖入页面

## 文件结构

```
manifest.json    # 扩展配置文件
content.js       # 内容脚本，实现按钮功能
styles.css       # 按钮样式
pack.ps1         # 打包脚本（生成 refresh-button.zip）
README.md        # 说明文档
```

## 使用方法

- 拖动按钮：按住按钮并拖动到任意位置
- 刷新页面：点击按钮执行硬刷新
- 按钮会自动保持在可视区域内

## 技术细节

- 使用 Manifest V3
- 纯前端实现，无需后台脚本
- 支持 MutationObserver 监听单页应用变化
- 包含触摸设备支持

## 注意事项

- 按钮的 z-index 设置为 10000，确保显示在最上层
- 硬刷新会忽略浏览器缓存，重新加载所有资源
- 在某些受限制的页面可能无法正常工作
- 运行 `.\pack.ps1` 可重新打包为 zip
